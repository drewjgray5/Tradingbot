            Help & FAQ
            
      
              
      Why two “Connect” buttons?
      
              
      
                Schwab splits API access between market data (quotes, history) and your
                brokerage account (balances, positions, orders). This app needs both — that’s why
                steps 1 and 2 each require a separate Schwab approval.
              
      
              
      Before you start
      
              
        
                
      • Sign in to this site first.
        • 
                
      • Allow pop-ups / redirects — you’ll briefly leave to approve on Schwab.
        • 
                
      • Use the same Schwab login you use for brokerage (or paper trading).
        • 
              
      
              
      If something fails
      
              
        
                
      • Try the failing step again after a full page refresh.
        • 
                
      • Confirm both account and market approvals completed.
        • 
                
      • Check for an error banner after returning from Schwab; copy the message if you contact support.
        •
